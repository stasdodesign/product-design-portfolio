Dark Patterns Explorer

An interactive catalog of 72 real-world dark UX patterns found in modern digital products.

Search it. Filter it. Explore real cases.

See how interfaces shape user behavior.

🚨 What this is

This project is a structured, interactive database of dark UX patterns used in real products.

Each pattern includes:

Clear explanation of the mechanism
Real-world case studies (from actual products)
Interactive search & filtering
Visual, readable format for fast exploration

It is designed to help you recognize how digital products influence user decisions.

🧠 Why it exists

Most people use digital products every day without noticing how design influences their behavior.

This project helps you:

Spot manipulation patterns in interfaces
Understand how product decisions affect user actions
Develop critical UX awareness
Learn from real implementations, not theory
🔍 What you can do
Search across 72 patterns
Filter by category, platform, or type
Explore real case studies behind each pattern
Understand how companies apply these techniques in production
🧩 Examples of patterns included
Forced continuity
Hidden subscription traps
Misdirection in UI flows
Scarcity & urgency manipulation
Default opt-ins
Social pressure triggers

Each one is linked to real product behavior.

⚖️ Ethical note

This project is not about blaming products or companies.

It is about understanding how design influences human decision-making.

Awareness is the first step toward better, more ethical UX.
